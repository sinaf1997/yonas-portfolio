# Yonas Aklilu Mekonnen Portfolio

Clean personal portfolio website for Yonas Aklilu Mekonnen, built with React, Vite, Tailwind CSS, and Framer Motion.

## Sections

- Home
- About Me
- Education
- Skills
- Certifications
- Practical Experience
- Selected Projects
- Contact

## Theme

- Primary: Dark Blue `#0A2540`
- Secondary: Royal Blue `#2563EB`
- Accent: Light Blue `#60A5FA`
- Background: White and light gray, with optional dark mode

## Commands

```bash
npm install
npm run dev
npm run build
```
