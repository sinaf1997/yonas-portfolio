Add your profile image here so the Hero component can display it.

Filename: `profile.jpg`
Path: `src/assets/profile.jpg`

Recommended:
- Provide a square image (1:1) of at least 512x512 for good quality.
- Optionally add `profile.webp` for better compression and update the import in `src/app/components/Hero.tsx`.

If you prefer not to add an image, the Hero component will show the initials fallback.

To update the image, replace the file at the path above and restart the dev server if needed.
